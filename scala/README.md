# Algolia's Scala playground


## Requirements

- JDK 8 or above
- Scala 2.12 or above
- SBT

## Using the playground

### From the IDE

We recommend you to use [IntellijIDEA](https://www.jetbrains.com/idea/).

Open `Main.scala` in Intellij and run the Main object.

### From sbt

To run the playground from sbt:


```bash
sbt compile

sbt run
```
